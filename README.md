# Chat-App

## Setup .env file
```
MONGODB_URI=...
PORT=5001
JWT_SECRET=...

# To use LLM capabilities
OPENAI_API_KEY=...

# To integrate with Jira
JIRA_API_KEY=...
JIRA_DOMAIN=...
JIRA_PROJECT_KEY=...
```
## Run Backend Node Server
```
cd backend
npm install
npm run dev
```
## Run Frontend React Server
```
cd frontend
npm install
npm run dev
```
